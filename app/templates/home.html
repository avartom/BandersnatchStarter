Health Insurance Claim Predictor

This application predicts the health insurance claims based on a number of demographic variables including age, gender, BMI(body mass index), bloodpressure, diabetis, smoking and region. The prediction is based on more than 1300 rows. In addition, it uses on a Random Forest Classifier machine learning algorithm.

Tech Stack

  • Logic: Python3
  • API Framework: Flask
  • Templates: Jinja2
  • Structure: HTML5
  • Styling: CSS3
  • Database: MongoDB
  • Graphs: Altair
  • Machine Learning: Scikit
